# DecodeLabs-Internship

## Project 1 - Data Cleaning and Preparation

### Objective

In this project, I cleaned an e-commerce dataset by handling missing values, checking duplicates, and validating data formats.

### Tasks Performed

* Identified and handled missing values in the CouponCode column
* Verified duplicate rows and confirmed there were no duplicates
* Verified all Order IDs were unique
* Standardized date formats for consistency
* Checked text consistency and removed extra spaces
* Validated TotalPrice calculations

### Dataset Files

* Dataset for Data Analytics.xlsx (Raw Dataset)
* Cleaned_Dataset.xlsx (Cleaned Dataset)

### Tools Used

* Microsoft Excel

### Final Outcome

After cleaning and validation, the dataset is now accurate, consistent, and ready for further analysis.
