
# DecodeLabs-Internship

## Project - 3 SQL Data Analysis Project

## 📌 Project Overview

This project is part of the DecodeLabs Data Analytics Internship Program.

The main objective of this project is to analyze a dataset using SQL queries and extract meaningful insights from raw data.

In this project, SQL is used to filter, sort, group, and perform calculations on the dataset.

---

## 🎯 Project Objective

The goal of this project is:

- To understand SQL Data Analysis
- To write SQL queries for extracting information
- To perform data filtering and grouping
- To generate insights using SQL functions

---

## 📂 Dataset Information

Dataset Name:

**Dataset for Data Analytics**

The dataset contains order-related information such as:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Coupon Code
- Referral Source
- Total Price

## 🔍 SQL Concepts Used

The following SQL concepts were implemented:

### 1. SELECT
Used to retrieve data from the database.

### 2. WHERE
Used to filter specific records based on conditions.

### 3. ORDER BY
Used to arrange data in ascending or descending order.

### 4. GROUP BY
Used to group similar data categories.

### 5. Aggregate Functions

Functions used:

- COUNT()
- SUM()
- AVG()
