# Project - 4 Power BI Sales Dashboard

## 📌 Project Overview

This project is part of the DecodeLabs Data Analytics Internship Program.

The main objective of this project is to analyze an E-commerce Sales dataset using Microsoft Power BI and create an interactive dashboard to visualize business insights.

The project includes data cleaning, data transformation, dashboard creation, and visualization for better business decision-making.

---

## 🎯 Project Objective

The goal of this project is:

- To understand Power BI Data Visualization.
- To clean and transform raw data using Power Query.
- To create an interactive dashboard.
- To analyze sales performance.
- To generate meaningful business insights.

---

## 📂 Dataset Information

**Dataset Name:**

Dataset for Data Analytics (3).xlsx

The dataset contains order-related information such as:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source
- Total Price

## 🧹 Data Cleaning Performed

The following data cleaning steps were performed:

- Imported Excel dataset into Power BI.
- Promoted column headers.
- Corrected data types.
- Converted Date column into Date format.
- Replaced null values in CouponCode with "No Coupon".
- Removed duplicate records.
- Validated TotalPrice values.
- Loaded cleaned data into Power BI.
